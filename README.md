# PT. AYARI KREASI PERSADA

## Company Profile & Custom Product Ordering Platform

A web-based company profile and custom product ordering platform developed using Laravel and Tailwind CSS.

---

## ✨ Features

- Company Profile
- Product Showcase
- Portfolio (Our Work)
- Send Your Project
- Authentication
- Role Management (Admin, Client, QC)
- Production Tracking
- Order Management
- Responsive Design

---

## 🛠️ Technologies

- Laravel
- PHP
- Tailwind CSS
- JavaScript
- MySQL
- Vite
